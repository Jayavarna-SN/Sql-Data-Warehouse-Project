## ğŸ—ï¸ Data Architecture

This project follows the Medallion Architecture with three layers:

![Data Architecture](https://github.com/Jayavarna-SN/Sql-Data-Warehouse-Project/blob/47556366f88e2838ea41484583c10ddcaf8b2443/DATA%20ARCHITECTURE.png)

**1. Bronze Layer** - Stores raw data directly from CSV files.Data is loaded into SQL Server without any transformation.


**2. Silver Layer** - Cleans and standardizes the data.Removes duplicates, fixes data types, and handles missing values.


**3. Gold Layer** - Contains business-ready data.Data is modeled into fact and dimension tables (star schema).Used for reporting and analysis.
